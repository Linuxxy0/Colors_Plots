# SciVizLab Project Overview

SciVizLab 是一个面向科研展示场景的前端可视化模板，目标很明确：

1. 提供适合科研页面的主题配色
2. 提供轻量但清晰的图表预览
3. 让默认样例数据和用户上传数据都能直接驱动界面

## 当前版本能力

- 结构化首页
- 3 套主题配色
- 6 类图表预览
- CSV / JSON 本地上传
- 全站共享上传数据状态
- GitHub Pages 自动部署

## 页面结构

- `HomePage`：项目入口和总览
- `PalettesPage`：主题浏览与真实界面预览
- `ChartsPage`：图表目录与大图预览
- `PlaygroundPage`：上传数据并切换图表类型

## 设计取向

这个项目偏向“展示模板”而不是“完整分析平台”：

- 图表预览使用轻量自绘实现，适合演示和作品展示
- 数据上传在浏览器端完成，不需要后端
- 适合直接部署到 GitHub Pages

## 推荐使用方式

- 作为 GitHub 项目展示页
- 作为科研或课程作品首页
- 作为数据可视化原型模板
- 作为后续接入正式图表库前的前端基础
