# Project Overview

SciVizLab 是一个围绕“科研风格展示”设计的前端项目。

## 页面结构

1. Hero 首屏：突出科研配色与多图表预览
2. Highlights：概括核心能力
3. Theme Showcase：展示三套主题配色
4. Chart Gallery：展示多图表类型
5. Dashboard Demo：展示完整研究型面板
6. Use Cases：呈现场景落点
7. Quick Start：展示安装与最小示例
8. Docs / Roadmap：强化开源项目感

## 设计目标

- 看起来像成熟的科研产品
- 适合截图进作品集
- 适合放在 GitHub 仓库主页演示
- 能继续扩展成完整数据可视化系统
